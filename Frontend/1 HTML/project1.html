Biography

Linus torvalds is a finish software engineer who is created and lead developer of the linux kernel used by linux distributions like debian arch and android .He also created the distribution version control system git.

Achievements

  • 1991- created the linux and the kernel version control system git
  • 2004- named one of the "100 most influential people in the world "
  • 2010-recieved an honorary millennium technology prize and the 2012 free software foundation
